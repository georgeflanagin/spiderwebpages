====================================
healpy
====================================


Description
~~~~~~~~~~~~

``healpy`` is a Python package to handle pixelated data on the sphere.
It is based on the Hierarchical Equal Area isoLatitude Pixelization
(HEALPix) scheme and bundles the HEALPix C++ library.
